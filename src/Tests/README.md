# Tests

ONE-vscode's unittests are based on [mocha](https://mochajs.org/).

## How to run tests?

```
ONE-vscode$ npm test
```

or

```
ONE-vscode$ ./node_modules/mocha/bin/mocha ./out/**/*.test.js
```

## How to add tests?

https://www.chaijs.com/api/assert/
